## Consists:
* Backend:
   * Laravel 10
   * Laravel Sanctum
   * Laravel auth/ui
   * maatwebsite/excel
* Frontend admin:
    * Laravel Blade
    * AdminLTE
    * Bootstrap 5
* Frontend client: 
    * Vue 3
    * Vuex
    * Bootstrap 5

## Includes:
* Admin-panel with crud main entites of app and some statistics on main page
* Client-side application on Vue 3 with basic functions (cart, catalog with filters, personal page with account data)
